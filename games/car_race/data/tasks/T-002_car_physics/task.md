# Task: T-002_car_physics

## Description
Create a Car class in game.js. Initialize two car instances for Player 1 and Player 2 with different controls. 
Implement basic forward/backward movement and rotation logic based on the controls.

## Acceptance Criteria
- A Car class handles position (x, y), angle, speed, max speed, acceleration, and friction.
- Two cars are instantiated and drawn as differently colored rectangles on the canvas.
- Player 1 uses WASD (W/S for forward/backward, A/D for left/right rotation).
- Player 2 uses Arrow Keys (Up/Down for forward/backward, Left/Right for rotation).
- Cars move and rotate correctly according to the pressed keys.
