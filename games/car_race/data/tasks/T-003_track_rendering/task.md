# Task: T-003_track_rendering

## Description
Draw a simple race track with inner and outer boundaries, plus a start/finish line. Add logic so cars are slowed down drastically when off the track (on grass/gravel).

## Acceptance Criteria
- A track shape is drawn on the canvas (e.g. an oval or figure-8).
- The track has a distinct color from the "off-track" background (grass).
- A finish line is visible on the track.
- Logic is implemented to detect if a car's center (x, y) is inside the track boundaries.
- If a car is "off-track", its speed and acceleration are greatly reduced to simulate driving on grass.
