# Task: T-004_car_collision

## Description
Implement 2D collision detection and response between the two cars so they bump into each other and deflect.

## Acceptance Criteria
- Cars cannot easily pass through each other.
- When they touch, there is a visual/physical deflection (they bounce apart based on their velocity).
- Use a simple geometric collision shape (e.g. circles or AABB) for the cars if true OBB is too complex.
