# INDEX

## Context
This project is a top-down 2D, 2-player car racing game made using HTML5 and JavaScript (Canvas API).

## Technical Stack
- HTML5 Canvas for rendering.
- Vanilla JavaScript for game logic and physics.
- CSS for simple layout.
