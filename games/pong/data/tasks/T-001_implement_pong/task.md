# Task: T-001_implement_pong

## Goal
Implement a playable, local HTML5 Canvas version of the classic game Pong.

## Requirements
- Create index.html, style.css (optional, can be inline), and script.js (optional, can be inline).
- The canvas should have a black background and white elements (paddles, ball, center dashed line, score).
- The player controls the left paddle using Up/Down arrow keys or W/S keys.
- The right paddle is controlled by a simple AI that tracks the ball's Y position.
- The ball should bounce off the top and bottom walls and both paddles.
- When the ball passes a paddle, the opposite side gains a point and the ball resets to the center.

## Definition of Done
- All files are created and placed in the root directory.
- The game can be played locally by opening index.html in a web browser.
- The score increments correctly.
- Code must be clean, readable, and well-commented.
