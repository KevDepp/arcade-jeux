# SPEC

Context:
- The user requested to create the game Pong, the first video game created. We will build a simple, playable 2D Pong game.

Requirements:
- Technology: HTML5 Canvas, CSS, JavaScript (Local web application).
- Gameplay: Player vs AI.
- Score tracking: Display the score for both the player and the AI.
- Visuals: Classic black background, white paddles and ball, dashed line in the middle.

Acceptance criteria:
- An index.html file exists and can be opened in a browser.
- The game loop runs smoothly.
- Player can move their paddle up and down using arrow keys or W/S.
- AI paddle follows the ball.
- Ball bounces off top and bottom edges, and off both paddles.
- Score increments when the ball passes a paddle.
